# Руководство по внесению вклада

Спасибо за интерес к проекту! Мы приветствуем любые вклады.

## Как внести вклад

1. **Fork** репозитория
2. Создайте **ветку** для вашей функции (`git checkout -b feature/AmazingFeature`)
3. **Закоммитьте** изменения (`git commit -m 'Add some AmazingFeature'`)
4. **Push** в ветку (`git push origin feature/AmazingFeature`)
5. Откройте **Pull Request**

## Стандарты кода

- Следуйте стилю кода Python (PEP 8)
- Добавляйте комментарии к новым функциям
- Обновляйте документацию при необходимости
- Тестируйте изменения перед отправкой PR

## Сообщения о проблемах

При создании issue, пожалуйста:
- Используйте понятное название
- Опишите проблему подробно
- Укажите шаги для воспроизведения
- Укажите версию Python и PostgreSQL

## Вопросы

Если у вас есть вопросы, создайте issue с меткой "question".
