# Быстрая настройка проекта

## Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/yourusername/booking-system.git
cd booking-system
```

## Шаг 2: Создание виртуального окружения

```bash
python -m venv venv
```

### Активизация виртуального окружения

**Linux/Mac:**
```bash
source venv/bin/activate
```

**Windows:**
```bash
venv\Scripts\activate
```

## Шаг 3: Установка зависимостей

```bash
pip install -r requirements.txt
```

## Шаг 4: Настройка базы данных

1. Убедитесь, что PostgreSQL установлен и запущен

2. Создайте базу данных:
```sql
CREATE DATABASE booking_system;
```

3. Создайте файл `.env`:
```bash
cp .env.example .env
```

4. Отредактируйте `.env` и укажите ваши параметры подключения:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=booking_system
DB_USER=postgres
DB_PASSWORD=your_password
```

## Шаг 5: Инициализация таблиц

```bash
python backend.py
```

Вы должны увидеть сообщение об успешном создании всех таблиц.

## Шаг 6: Запуск приложения

```bash
python main.py
```

или

```bash
python gui.py
```

## Проверка установки

После запуска должно открыться окно приложения с вкладками:
- Пользователи
- Столы
- Бронирования
- Проверка доступности

Если все работает корректно, установка завершена успешно!

## Возможные проблемы

### Ошибка подключения к базе данных
- Проверьте, что PostgreSQL запущен
- Убедитесь, что параметры в `.env` корректны
- Проверьте права пользователя на доступ к базе данных

### Модуль не найден
- Убедитесь, что виртуальное окружение активировано
- Переустановите зависимости: `pip install -r requirements.txt`

### Ошибка при создании таблиц
- Убедитесь, что база данных существует
- Проверьте права пользователя на создание таблиц
